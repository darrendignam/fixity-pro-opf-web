What Next for Existing Subscribers.

Existing subscribers no longer need to pay and should not renew their subscription. The software will continue to work as it has done previously.

  • It is no longer necessary to subscribe/pay to use Fixity Pro.
  • Existing subscriptions should continue to operate without any action of the subscriber's part.
  • The software will still try to validate a subscription on startup, this check should never fail.
  • OPF will produce a new version of the software with all subscription features disabled.
